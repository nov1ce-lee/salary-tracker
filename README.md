# Salary Tracker

实时薪资追踪器——精确计算每一秒的价值。

## 功能

- **实时工资计算** — 配置上下班时间后，每秒自动刷新今日/本月累计额
- **工作日智能识别** — 自动排除周末和中国法定节假日
- **加班日标记** — 点击日历格子即可标记加班日（含周末），实时计入工资
- **多端适配** — 响应式布局，手机 / 平板 / 桌面均可使用
- **本地持久化** — 配置和加班标记自动保存在浏览器 localStorage

## 计算公式

```
基础日薪 = 基础月薪 ÷ 月法定工作日
月实际工资 = 基础日薪 × (月法定工作日 + 加班天数)
今日累计 = 基础日薪 × (已工作分钟 ÷ 总工作分钟)
```

## 快速开始

```bash
# 直接用浏览器打开
open index.html

# 或启动本地服务器
npx serve -p 3000
```

## 部署

本项目配置了 GitHub Actions 自动部署到 GitHub Pages：

1. 在 GitHub 仓库 Settings → Pages 中启用 GitHub Actions
2. 推送到 `main` 分支即可自动构建并部署
